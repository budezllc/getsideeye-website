Support

Side Eye is a local-first Chrome extension. Most issues relate to LM Studio, Ollama, or browser permissions.

Contact

Email: hello@getsideeye.com

Before you write

  • Chrome version (need 114+ for the side panel)
  • Side Eye version (from chrome://extensions)
  • AI profile in use (LM Studio or Ollama) and whether the local server is running
  • Steps to reproduce and any error text from the side panel

Common fixes

  • No reply / connection error — confirm LM Studio or ollama serve is running and the server URL in Options matches.
  • CORS / 403 from local server — Side Eye is a Chrome extension and needs CORS enabled on your AI server. LM Studio: Local Server → Server Settings → turn on CORS. Ollama: set OLLAMA_ORIGINS to *, then restart Ollama. On Windows, add it under Environment Variables; on macOS, run launchctl setenv OLLAMA_ORIGINS "*" and quit/reopen the Ollama app.
  • Page context empty — enable the eye icon; check the domain is not on your ignore list.
  • Voice not working — enable Speech in Options and wait for the model download to finish; try Re-download models if needed.
  • Extension not visible — click the puzzle icon in Chrome and pin Side Eye; then open the side panel from the toolbar icon.

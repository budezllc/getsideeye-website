Features

Built for browsing with a local LLM

Everything runs where you already keep your models — on your machine, in your browser.

Side panel chat

Open Side Eye beside any tab. Multiple chat tabs, streaming markdown replies, and a profile picker for LM Studio or Ollama.

Page context

Toggle the eye to include URL, selection, and readable page text — only when your message needs it. Ignore sensitive domains entirely.

Vision & images

Attach up to eight images per message. Vision works with capable local models through LM Studio or Ollama’s native APIs.

Quick actions

Summarize, find related sites, or run custom prompts from the context bar. Starter chips adapt to articles, docs, social, and more.

On-device voice

Optional read-aloud with Supertonic TTS. Models download once; synthesis runs in the browser with a reactive orb control.

MCP tools via LM Studio

Give local models real capabilities through LM Studio’s MCP support — plug in tools like Brave Search for live internet access, so your on-device LLM can reach beyond its training data.

Your settings

Dark or light theme, font size, privacy ignore lists, and per-profile temperature and context length.

How it works

Three steps to a side glance

  1. 1

    Run a local model

    Start LM Studio or ollama serve on your machine. Enable CORS in LM Studio server settings, or set OLLAMA_ORIGINS for Ollama. Side Eye talks to localhost — no API keys required for local profiles.

  2. 2

    Install the extension

    Add Side Eye from the Chrome Web Store, open the side panel from the toolbar, and pick your AI profile in the footer.

  3. 3

    Chat with context

    Turn on page context when you want the current tab included. Ask naturally — “summarize this page”, “explain this selection”, or paste images.

Privacy

Your data stays yours

  • Chat history lives in chrome.storage.local on your device.
  • Page excerpts are read in-browser and sent only to your local model.
  • Ignored domains are never read — banking, mail, or anything you block in settings.
  • Optional TTS models download from Hugging Face once; speech runs on-device.

Install

Get Side Eye in Chrome

Side Eye requires Chrome 114+ for the Side Panel API. Install from the Chrome Web Store or build from source on GitHub.

Source + Docs on GitHub

Clone the repo, build with npm, and load the unpacked dist folder in Chrome — best for developers or the latest code.

  1. Clone the repo and run npm install and npm run build.
  2. Open chrome://extensions and enable Developer mode.
  3. Click Load unpacked and select the dist folder.
View on GitHub

Chrome Web Store

The fastest way to install — no build step, no developer mode. Free from the Chrome Web Store.

  1. Click Add to Chrome on the Chrome Web Store listing.
  2. Confirm the install when Chrome prompts you.
  3. Pin Side Eye from the extensions menu, then configure LM Studio or Ollama in Options.
